# Объяснение работы Telegram бота

## Почему бот останавливается?

В файле `examples/telegram_bot_advanced.sup` бот останавливается после 50 итераций по следующим причинам:

### 1. Демонстрационный режим

Бот работает в **демонстрационном режиме** для тестирования. Это сделано специально, чтобы:

- Показать работу всех команд
- Не создавать бесконечный процесс при тестировании
- Продемонстрировать симуляцию входящих сообщений

### 2. Код с ограничением

```sup
func botLoop() -> int {
    var iteration = 0
    
    # Ограничение для демонстрации
    while iteration < 50 {
        # ... обработка сообщений ...
        iteration = iteration + 1
    }
    
    return 0
}
```

Условие `while iteration < 50` останавливает бот после 50 итераций.

## Как сделать бота бесконечным?

### Вариант 1: Бесконечный цикл с флагом

```sup
func botLoop() -> int {
    var iteration = 0
    var running = 1  # Флаг работы бота
    
    # Бесконечный цикл
    while running == 1 {
        getUpdates()
        # ... обработка сообщений ...
        iteration = iteration + 1
    }
    
    return 0
}
```

В этом случае бот будет работать **бесконечно**, пока:
- Не нажмете Ctrl+C в консоли
- Не закроете программу
- Не установите `running = 0` в коде

### Вариант 2: Условие true (когда будет поддержка)

```sup
func botLoop() -> int {
    # Бесконечный цикл
    while true {
        getUpdates()
        # ... обработка сообщений ...
    }
    
    return 0
}
```

## Обновленная версия

В обновленной версии `telegram_bot_advanced.sup` бот теперь:

1. **Работает бесконечно** с флагом `running`
2. **Останавливается после 50 итераций** только для демонстрации
3. **Показывает предупреждение** о том, что это демонстрационный лимит

### Код обновленного бота:

```sup
func botLoop() -> int {
    print("🤖 Бот запущен и готов к работе!")
    print("💡 Для остановки бота нажмите Ctrl+C\n")
    
    var iteration = 0
    var running = 1
    
    # Бесконечный цикл - бот работает постоянно
    while running == 1 {
        getUpdates()
        
        # ... обработка сообщений ...
        
        # Для демонстрации остановим после 50 итераций
        # В реальном боте эту проверку нужно убрать
        if iteration >= 50 {
            print("\n⚠️  Достигнут лимит демонстрации (50 итераций)")
            print("💡 В реальном боте эта проверка должна быть удалена")
            running = 0
        }
        
        iteration = iteration + 1
    }
    
    return 0
}
```

## Для реального Telegram бота

Чтобы создать **настоящий** Telegram бот, который работает постоянно:

### 1. Удалите проверку лимита

```sup
func botLoop() -> int {
    var iteration = 0
    var running = 1
    
    while running == 1 {
        getUpdates()
        
        # Обработка реальных сообщений из Telegram API
        # (вместо симуляции)
        
        iteration = iteration + 1
    }
    
    return 0
}
```

### 2. Подключите реальный Telegram API

Замените симуляцию на настоящие HTTP запросы:

```sup
func getUpdates() -> int {
    var url = API_URL + BOT_TOKEN + "/getUpdates"
    
    if last_update_id > 0 {
        url = url + "?offset=" + str(last_update_id)
    }
    
    # Здесь должен быть реальный HTTP запрос
    # Например, через библиотеку curl или http
    
    return 0
}
```

### 3. Добавьте обработку ошибок

```sup
func botLoop() -> int {
    var running = 1
    var errors = 0
    
    while running == 1 {
        var result = getUpdates()
        
        if result != 0 {
            errors = errors + 1
            print("Ошибка при получении обновлений")
            
            # Если слишком много ошибок, остановить бота
            if errors > 10 {
                print("Слишком много ошибок, остановка бота")
                running = 0
            }
        }
    }
    
    return 0
}
```

## Запуск бота

### Демонстрационный режим (текущий):

```bash
./sup.exe dev examples/telegram_bot_advanced.sup
```

Бот остановится после 50 итераций.

### Бесконечный режим:

1. Откройте `examples/telegram_bot_advanced.sup`
2. Найдите строки:
   ```sup
   if iteration >= 50 {
       running = 0
   }
   ```
3. Удалите или закомментируйте эти строки
4. Запустите бота:
   ```bash
   ./sup.exe dev examples/telegram_bot_advanced.sup
   ```
5. Для остановки нажмите **Ctrl+C**
