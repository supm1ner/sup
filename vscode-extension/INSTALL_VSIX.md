# Установка расширения Sup Language Support

## Быстрая установка

### Способ 1: Установка из VSIX файла

1. Откройте VS Code
2. Нажмите `Ctrl+Shift+P` (или `Cmd+Shift+P` на Mac)
3. Введите "Extensions: Install from VSIX"
4. Выберите файл `sup-language-0.2.0.vsix` из папки `vscode-extension/`
5. Перезапустите VS Code

### Способ 2: Через командную строку

```bash
code --install-extension vscode-extension/sup-language-0.2.0.vsix
```

## Что включено

### ✅ Подсветка синтаксиса

- Ключевые слова: `func`, `var`, `const`, `if`, `else`, `while`, `for`, `return`
- Типы данных: `int`, `float`, `string`, `bool`, `void`
- Операторы: `+`, `-`, `*`, `/`, `==`, `!=`, `<`, `>`, `and`, `or`, `not`
- Литералы: числа, строки, булевы значения
- Комментарии: `# однострочные`

### ✅ Иконки файлов (NEW в v0.2.0!)

Файлы `.sup` теперь отображаются с собственной иконкой в проводнике VS Code!

**Активация иконок:**

1. Откройте настройки VS Code (`Ctrl+,`)
2. Найдите "File Icon Theme"
3. Выберите "Sup File Icons"

Или через командную палитру:
1. `Ctrl+Shift+P`
2. "Preferences: File Icon Theme"
3. Выберите "Sup File Icons"

### ✅ Автоматические возможности

- Автозакрытие скобок: `()`, `{}`, `[]`
- Автозакрытие кавычек: `""`, `''`
- Автоматические отступы
- Сворачивание кода (folding)
- Комментирование строк: `Ctrl+/`

## Проверка установки

1. Откройте любой файл `.sup` в VS Code
2. Проверьте, что синтаксис подсвечивается
3. Проверьте, что файл отображается с иконкой Sup в проводнике
4. Попробуйте закомментировать строку: `Ctrl+/`

## Примеры файлов

Откройте примеры из папки `examples/`:

- `hello.sup` - простой Hello World
- `calculator.sup` - калькулятор
- `telegram_bot_advanced.sup` - Telegram бот
- `library_test.sup` - тест библиотек

## Обновление расширения

Если вы обновили расширение:

1. Удалите старую версию:
   - `Ctrl+Shift+X` → найдите "Sup Language Support" → Удалить
2. Установите новую версию из VSIX файла
3. Перезапустите VS Code

## Возможные проблемы

### Проблема: Синтаксис не подсвечивается

**Решение:**
1. Убедитесь, что файл имеет расширение `.sup`
2. Проверьте, что расширение установлено: `Ctrl+Shift+X` → "Sup Language Support"
3. Перезапустите VS Code

### Проблема: Иконки не отображаются

**Решение:**
1. Откройте настройки: `Ctrl+,`
2. Найдите "File Icon Theme"
3. Выберите "Sup File Icons"
4. Перезапустите VS Code

### Проблема: Расширение не устанавливается

**Решение:**
1. Проверьте версию VS Code (требуется 1.60.0 или выше)
2. Попробуйте установить через командную строку:
   ```bash
   code --install-extension vscode-extension/sup-language-0.2.0.vsix
   ```

## Сборка расширения из исходников

Если вы хотите собрать расширение самостоятельно:

```bash
cd vscode-extension

# Установить vsce (если еще не установлен)
npm install -g @vscode/vsce

# Собрать VSIX
vsce package
```

Это создаст файл `sup-language-X.X.X.vsix`.
