# Changelog

Все изменения в расширении Sup Language Support.

## [0.2.0] - 2024-01-XX

### Добавлено
- Иконки для файлов .sup в проводнике VS Code
- Тема иконок файлов "Sup File Icons"
- Поддержка отображения иконки языка в редакторе

### Улучшено
- Обновлена документация
- Улучшена интеграция с VS Code

## [0.1.0] - 2024-01-XX

### Добавлено
- Подсветка синтаксиса для языка Sup
- Поддержка всех ключевых слов (func, var, const, if, else, while, for, return, и т.д.)
- Подсветка типов данных (int, float, string, bool, void)
- Подсветка операторов и литералов
- Поддержка комментариев (# однострочные)
- Автозакрытие скобок, кавычек
- Автоматические отступы
- Сворачивание кода (folding)
- Комментирование строк (Ctrl+/)
- Иконка расширения
