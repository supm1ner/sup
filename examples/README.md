# Примеры программ на языке Sup

Эта директория содержит примеры программ, демонстрирующих возможности языка Sup.

## Список примеров

### 1. Hello World (`hello.sup`)
Простейшая программа "Hello World".

**Запуск:**
```bash
./sup.exe dev examples/hello.sup
```

**Что демонстрирует:**
- Объявление функции `main`
- Переменные с автоматическим выводом типа
- Встроенная функция `print()`

---

### 2. Математические операции (`math.sup`)
Демонстрация базовых математических операций.

**Запуск:**
```bash
./sup.exe dev examples/math.sup
```

**Что демонстрирует:**
- Функции с параметрами и типами
- Арифметические операции
- Вызов функций
- Работа с числами (int и float)

---

### 3. Калькулятор (`calculator.sup`)
Полнофункциональный калькулятор с различными операциями.

**Запуск:**
```bash
./sup.exe dev examples/calculator.sup
```

**Что демонстрирует:**
- Сложение, вычитание, умножение, деление
- Возведение в степень
- Вычисление процентов
- Условные операторы (if)
- Циклы (while)
- Конкатенация строк
- Преобразование чисел в строки с помощью `str()`

**Функции:**
- `add(a, b)` - сложение
- `subtract(a, b)` - вычитание
- `multiply(a, b)` - умножение
- `divide(a, b)` - деление (с проверкой деления на ноль)
- `power(base, exp)` - возведение в степень
- `percentage(value, percent)` - вычисление процента

## Встроенные функции

### `print(...)`
Выводит значения в консоль.

```sup
print("Hello, World!")
print("Число: " + str(42))
```

### `str(number)`
Преобразует число в строку.

```sup
var age = 25
print("Мне " + str(age) + " лет")
```

## Типы данных

- `int` - целые числа
- `float` - числа с плавающей точкой
- `string` - строки
- `bool` - логические значения (true/false)
- `void` - отсутствие значения

## Синтаксис

### Объявление функций
```sup
func имя_функции(параметр1: тип, параметр2: тип) -> тип_возврата {
    # тело функции
    return значение
}
```

### Переменные
```sup
var имя = значение          # автоматический вывод типа
var имя: тип = значение     # явное указание типа
const имя = значение        # константа
```

### Условные операторы
```sup
if условие {
    # код
} else {
    # код
}
```

### Циклы
```sup
while условие {
    # код
}

for переменная in диапазон {
    # код
}
```

## Компиляция

Все примеры можно скомпилировать в исполняемые файлы:

```bash
# Компиляция
./sup.exe compile examples/calculator.sup

# Запуск скомпилированной программы
./calculator.exe
```

## Режим разработки

Для быстрого тестирования используйте команду `dev`:

```bash
./sup.exe dev examples/calculator.sup
```

Эта команда автоматически компилирует и запускает программу.
